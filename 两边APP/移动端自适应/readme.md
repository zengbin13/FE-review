## 关于1px问题

- **css像素**：如在chrome中 iPhone6 `375*667`
- **物理像素**：在 iPhone6 包装上 `750*1334`

`设备像素比dpr = 物理像素/css像素 = 分辨率 / 屏幕尺寸`

1px问题：移动端某些手机 `1px`实际效果更粗，在UI设计稿中1px为物理像素，转换在手机显示上位 `0.5px`css像素。但是某些浏览器解析 `0.5px`时转换为 `1px`
